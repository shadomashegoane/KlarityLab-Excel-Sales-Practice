# KarityLab-Excel-Sales-Practice
The exercises move from basics to analysis: Section A covers totals and averages, Section B uses count functions, Section C applies COUNTIF and SUMIF, Section D focuses on the PRODUCT function, and Section E develops deeper analysis like top sales and quantities.

## 📂 Contents
- **data/** → Raw Excel practice file (`01.Excel_Sales_Practice.xlsx`)
- **exercises/** → Step-by-step practice questions organized by topic
- **solutions/** → Example solutions for reference


## 🎯 Learning Goals
- Perform **basic calculations** (SUM, AVERAGE, etc.)
- Apply **COUNT, COUNTA, COUNTIF, SUMIF** functions
- Use the **PRODUCT function** for sales calculations
- Conduct **data analysis** (highest sales, transactions above thresholds, etc.)
- Build confidence in handling **real-world datasets**


## 📝 Example Exercises
1. Calculate **Total Sales** for each row.  
2. Find the **Grand Total Sales**.  
3. Count how many sales reps are from the **West region**.  
4. Calculate **total sales for West region only** using `SUMIF`.  
5. Identify which product generated the **highest sales**.
